# 💻 Desafio: Primeiros Passos com HTML

Este repositório contém a resolução do desafio prático do **Módulo 1** da trilha de HTML da [Dio.me](https://dio.me). O objetivo foi construir a estrutura de um website utilizando as tags fundamentais de marcação.

## 🎯 Objetivo do Projeto
Desenvolver uma página web básica que demonstre o domínio de tags de texto, listas e links, aplicando conceitos de semântica e formatação.

## 🛠️ Tags Aplicadas

### Tags de Aula:
- **Títulos:** `<h1>` a `<h6>`
- **Formatação de Texto:** `<p>`, `<mark>`, `<small>`, `<i>`, `<u>`, `<strong>`
- **Listas:** `<ol>`, `<ul>`, `<li>`
- **Elementos Especiais:** `<a>`, `<hr>`, `<sub>`, `<sup>`, `<blockquote>`

### Tags de Pesquisa (Desafio Extra):
- `<font>` (Estilização legada)
- `<del>` (Texto removido)
- `<abbr>` (Abreviações com auxílio de tooltip)

## 🚀 Como Visualizar
1. Faça o clone do repositório:
   ```bash
   git clone https://github.com
   ```
2. Abra o arquivo `index.html` em qualquer navegador.
